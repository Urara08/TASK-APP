## CLIのタスクアプリ
```
言語         :Rust
制作物       :タスク管理アプリ
制作期間　    :３日位
目的         :CLIでアプリを作り、学習した値を渡していく処理を行うことです。
難しかった所  :所有権とライフタイムの扱い、生データを型として扱うこと、配列に対してのツールの扱いです。
その他       :検索と並行して、難しかった点や検索での不明点についてはChat－GPTにヒントや答えを貰いながら進めました。
今後         :変数や関数で処理が分かりやすいようにしたいです。
```

> 使い方
* cargo run でスタート。
* 未完了のタスクがある場合は表示されます。
* 0:登録か1:削除を数字で入力。
* 登録⇒登録したいタスクを入力。
* 完了⇒削除したいタスク番号を入力。
